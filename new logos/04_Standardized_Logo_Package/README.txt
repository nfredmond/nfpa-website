Nat Ford Planning — Brand Package

Contents
--------
This package includes standardized PNG logo files in:
- light mode
- dark mode
- transparent background
- background-preserved versions
- multiple sizes

Naming convention
-----------------
primary_stacked       = main stacked logo
horizontal            = wide horizontal logo
square_monogram       = square NF monogram
circular_monogram     = circular NF monogram
monochrome_primary    = single-color primary logo

Folders
-------
light/background/     = original light-background PNGs
light/transparent/    = transparent-background PNGs
dark/background/      = original dark-background PNGs
dark/transparent/     = transparent-background PNGs

Recommended use
---------------
- Use transparent versions for websites, documents, and overlays.
- Use light versions on white or light backgrounds.
- Use dark versions on black or dark backgrounds.
- Use monochrome versions where a simpler one-color treatment is needed.

Note
----
These are high-resolution PNG assets generated from the approved logo variations.
If you'd like, the next refinement step would be:
1. vector redraw (SVG / PDF / EPS),
2. favicon/app icon exports,
3. social profile image crops,
4. business card / letterhead mockups.
